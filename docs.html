Install CLI

Requirements

  • Windows 10, Mac OSX, Linux
  • Node.js version 12 or higher

Install

Mac (Homebrew)

$ brew install Videoweb-Cli

NPM (Homebrew)

$ npm install Videoweb-Cli

Yarn (Homebrew)

$ yarn install Videoweb-Cli
